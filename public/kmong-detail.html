FAQ
자주 묻는 질문
5개의 질문
Q 기획서나 디자인이 없어도 되나요?
네, 전혀 문제없습니다. 제품 정보와 특장점만 간단히 알려주시면, 기획부터 디자인까지 모두 완성해 드립니다.
Q 제작기간은 얼마나 걸리나요?
기획안 확정 후 평균 7~14영업일 소요됩니다. 꼼꼼하게 기획하고 제작하기 때문에 물리적인 시간이 필요합니다.
Q 수정은 몇 번이나 가능한가요?
텍스트·이미지 교체 같은 간단한 수정은 런칭 후 30일 이내 횟수 제한 없이 무료입니다.
Q 소스코드는 어떻게 받나요?
완성된 프로젝트 전체를 zip 파일 또는 GitHub private repo로 전달해 드립니다. Next.js 기반 표준 프로젝트라 특정 빌더(워드프레스·아임웹 등)에 종속되지 않으며, 이후에도 다른 개발자가 이어받아 수정하실 수 있습니다.
Q 나중에 다른 호스팅으로 옮길 수 있나요?
네, 언제든 가능합니다. Vercel, Netlify, 카페24, AWS, 자체 서버 등 어느 환경으로든 이전할 수 있는 구조로 제작해 드립니다. 문의 관리에 사용되는 구글 시트 역시 고객님 구글 계정으로 소유권 이전이 가능하여, 장기적으로 저희에게 의존하지 않으셔도 됩니다.
💡
안내 — 도메인 구매 비용은 별도(연 1~2만원 내외)이며, 호스팅 비용은 사용 서버에 따라 달라질 수 있습니다. 정확한 견적은 상담 후 안내드립니다.
